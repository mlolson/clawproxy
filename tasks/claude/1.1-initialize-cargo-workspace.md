# Task 1.1: Initialize Cargo Workspace

**Assignee:** Claude
**Dependencies:** None
**Status:** Open

## Description

Set up the initial Rust project structure with Cargo, including two binary targets.

## Acceptance Criteria

- [ ] Run `cargo new clawproxy` or create Cargo.toml manually
- [ ] Configure Cargo.toml with two binaries:
  - `clawproxy` at `src/bin/clawproxy.rs`
  - `clawproxy-run` at `src/bin/clawproxy_run.rs`
- [ ] Set up directory structure:
  ```
  src/
  ├── bin/
  │   ├── clawproxy.rs
  │   └── clawproxy_run.rs
  ├── proxy/
  ├── sandbox/
  └── lib.rs
  ```
- [ ] Add all dependencies from IMPLEMENTATION_PLAN.md
- [ ] Verify: `cargo check` passes
