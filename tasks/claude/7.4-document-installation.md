## Installation

### macOS (Homebrew)

```bash
brew tap YOUR_ORG/clawproxy
brew install clawproxy
```

### Linux / macOS (Install Script)

```bash
curl -fsSL https://raw.githubusercontent.com/YOUR_ORG/clawproxy/main/install.sh | bash
```

Or with specific version:
```bash
CLAWPROXY_VERSION=0.1.0 curl -fsSL ... | bash
```

### From Source (Cargo)

Requires Rust 1.70+:
```bash
cargo install --git https://github.com/YOUR_ORG/clawproxy
```

### Manual Installation

1. Download the appropriate binary from [Releases](https://github.com/YOUR_ORG/clawproxy/releases)
2. Extract the archive
3. Move binaries to your PATH:
   ```bash
   sudo mv clawproxy clawproxy-run /usr/local/bin/
   ```
4. Make executable:
   ```bash
   chmod +x /usr/local/bin/clawproxy /usr/local/bin/clawproxy-run
   ```

## Quick Start

```bash
# Initialize configuration
clawproxy init

# Add your API keys
clawproxy secret set openai
clawproxy secret set anthropic

# Start the proxy
clawproxy start
```

## Verifying Installation

```bash
clawproxy --version
clawproxy-run --version
```

## Uninstalling

### Homebrew
```bash
brew uninstall clawproxy
```

### Manual
```bash
rm /usr/local/bin/clawproxy /usr/local/bin/clawproxy-run
rm -rf ~/.config/clawproxy
```

## Troubleshooting

### "command not found: clawproxy"

Ensure `/usr/local/bin` is in your PATH:
```bash
echo $PATH
```

Add to your shell profile if needed:
```bash
export PATH="/usr/local/bin:$PATH"
```

### Permission denied on Linux

You may need to install with sudo or use a user-writable location:
```bash
CLAWPROXY_INSTALL_DIR=~/.local/bin curl -fsSL ... | bash
export PATH="$HOME/.local/bin:$PATH"
```

### Sandbox not working on older Linux

Landlock requires Linux kernel 5.13+. Check your kernel version:
```bash
uname -r
```

On older kernels, clawproxy-run will warn and continue without sandboxing.
```
