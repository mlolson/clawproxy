# Task 1.2: Set Up Shared Library Structure

**Assignee:** Claude
**Dependencies:** 1.1
**Status:** Open

## Description

Create the shared library code structure with module organization and error types.

## Acceptance Criteria

- [ ] Create `src/lib.rs` exporting all modules:
  ```rust
  pub mod config;
  pub mod error;
  pub mod proxy;
  pub mod sandbox;
  ```
- [ ] Create `src/error.rs` with error types:
  - `Error` - main error enum
  - `ConfigError` - config-related errors
  - `ProxyError` - proxy-related errors
  - `SandboxError` - sandbox-related errors
- [ ] Create empty module files:
  - `src/config.rs`
  - `src/proxy/mod.rs`
  - `src/sandbox/mod.rs`
- [ ] Verify: `cargo check` passes

## Error Types Structure

```rust
// src/error.rs
use thiserror::Error;

pub type Result<T> = std::result::Result<T, Error>;

#[derive(Error, Debug)]
pub enum Error {
    #[error("Configuration error: {0}")]
    Config(#[from] ConfigError),

    #[error("Proxy error: {0}")]
    Proxy(#[from] ProxyError),

    #[error("Sandbox error: {0}")]
    Sandbox(#[from] SandboxError),

    #[error("IO error: {0}")]
    Io(#[from] std::io::Error),
}

#[derive(Error, Debug)]
pub enum ConfigError { ... }

#[derive(Error, Debug)]
pub enum ProxyError { ... }

#[derive(Error, Debug)]
pub enum SandboxError { ... }
```

## Notes

- Error types should cover all anticipated failure modes
- Use `thiserror` for derive macros
- Export `Result` type alias from lib.rs
